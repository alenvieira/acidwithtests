# AcidWithTests

Testes para explicar as propriedades do A.C.I.D.

## Configurando o ambiente

```console
python -m venv .acidwithtests
source .acidwithtests/bin/activate
pip install -r requirements.txt
```

## Como rodar os testes?

```console
source .acidwithtests/bin/activate
python -m unittest
```
